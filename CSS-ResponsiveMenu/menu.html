Things to Note

  • The responsive menu works by creating a list that contains an item called menu. This item is hidden on the full size version of the site by selecting the class and setting “display: none;”.
  • The “href” is necessary to remain consistant with the other menu items.
  • When the browser window is set less than 481 pixels the menu item is displayed as a block, and all of the other list items are hidden
  • the CSS “:hover” selector is used to change the display setting back to “display: block” in order to show all of the menu list elements.
  • Note that the cursor does not change to a text selection cursor when hovering over text. This is because “cursor: default” was set for the body
  • Note also the use of HTML special chars such ‘“’ and ‘”’ in this text.
